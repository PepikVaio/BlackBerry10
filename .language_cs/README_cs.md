# Diacritics CS!
Doplnění diakritiky do textu.

## Popis
Zkratka slouží k doplnění diakritiky do textu, který je uložený ve schránce.
* Jakmile se diakritika doplní, uloží se nový text do schránky. 
* testováno - iPhone
